# Blackjack
ohjelmani simuloi blackjack pelin toiminnot ja on tarkoitettu ajan tappamiseen ja hauskuuden pitoon.

## Käynnistäminen
 
1. Varmista että Python on asennettuna
2. Lataa tiedosto "peli"
3. Aja komennolla: python peli

## käyttö
suorita ohjelma ja vastaa terminaliin koodin kysymiin kysymyksiin pelattavaksesi.
